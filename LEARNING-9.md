# Learning Log

## Session 9 – Flask

### What I Learned
- Basics of Flask: creating a server and routes.
- Working with Jinja2 templates for dynamic HTML.
- Getting form data and passing it to the backend.
- Connecting frontend and backend for interactive web apps.

### What I Built
- Mini Flask app: user enters data in a form, server processes it, and returns results to the page.
- Interactive elements on the page controlled by both JS and Flask (saving data and dynamic display).

### Challenges & Solutions
- **Challenge:** passing form data to Python and back to the page.  
  **Solution:** used `request.form` and template rendering with variables.
- **Challenge:** interactive output without losing styling.  
  **Solution:** combined Flask and JS: Flask handles data, JS updates the page display.
